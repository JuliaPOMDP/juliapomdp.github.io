JuliaPOMDP

Packages for solving POMDPs in the Julia programming language.

What is JuliaPOMDP?

JuliaPOMDP is an organization that maintains software packages for defining and solving partially observable Markov decision processes (POMDPs). The POMDP formalism can be used to represent and solve a wide range of sequential decision problems encountered in the real world, for example autonomous driving and flying, planning logistical operations, and making medical diagnoses. JuliaPOMDP seeks to improve research, collaboration, and education related to POMDPs by bringing together a variety of problems and solution techniques in a unified software ecosystem.

JuliaPOMDP maintains packages written in the Julia programming language. Julia's combination of speed and flexibility make it uniquely suitable for both representing a large variety of problems and algorithms and providing the computational efficiency to solve large problems. The main package that JuliaPOMDP maintains is POMDPs.jl. This package provides an interface that makes problems and solvers interoperable as much as possible.
